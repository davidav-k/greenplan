GreenPlan - система планирования экологических проектов

О системе

GreenPlan - это комплексная система для планирования, управления и мониторинга экологических проектов. Система предоставляет инструменты для эффективного управления всеми аспектами экологических инициатив.

Основные возможности

  • Создание и управление экологическими проектами
  • Планирование ресурсов и бюджета
  • Мониторинг прогресса и результатов
  • Генерация отчетов и аналитика
  • Интеграция с внешними системами

Технологии

Система построена на современных технологиях:

  • Java 21 + Spring Boot 3
  • Thymeleaf для веб-интерфейса
  • PostgreSQL для хранения данных
  • Maven для сборки
